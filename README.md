# Student Management System 🎓

A Python CLI application for managing student records, built with SQLite.

## Features ✨
- ✅ Add new students
- 🔍 View all students
- 📝 Update student records
- ❌ Delete students
- 💾 Persistent database storage

## Prerequisites
- Python 3.9+


## Installation

### Method 1: Run directly with Python

# Clone the repository

# Run the application
